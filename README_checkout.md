##■デモアプリ
###①プロジェクトリポジトリをclone
###②Terminalから、プロジェクトのルートディレクトリで、以下のコマンドを実行するとアプリの起動が始まる
###     ./gradlew bootRun
###③「Started〜」メッセージが表示されれば、起動完了
###④ブラウザで、下記のURLへアクセルし、画面が表示されることを確認
###     http://localhost:8088/honest-info/demo/index
###⑤Ctrl + cで終了
